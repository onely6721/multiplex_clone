Web-app online booking system
</h1>

# Stack

- node.js
- express
- mongoDB
- react
- material ui


# Installation
```
git clone https://github.com/onely6721/multiplex_clone/

in server create .env file and add PORT, DB_URL, JWT_SECRET

cd server npm install && npm run dev
cd client npm install && npm start

```


# Run app
```
cd server npm run dev
cd client npm start
```

# Features!

- Add / Update / Delete Movies
- Add / Update / Delete Cinemas
- Add / Update / Delete Halls
- Add / Update / Delete / Showtimes
- Add / Update / Delete Reservations
- Add / Update / Delete Users
- Generate schedule for showtimes
